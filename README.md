# Ubuntu Sentinel

Ubuntu Sentinel is a PeaceTech MVP for the Andela x OpenAI Codex Accelerator. It helps conflict-affected communities turn raw local reports into structured accountability intelligence that can be validated by humans, adapted across OSF operational regions, and escalated as institution-ready briefs.

## MVP Focus

- Capture community reports through a low-bandwidth web experience.
- Preserve reports offline and sync them when connectivity returns.
- Localize the experience for OSF-relevant regions and languages.
- Use an AI-assisted, human-in-the-loop workflow to structure reports, compare commitments, estimate reparative needs, and draft accountability briefs.
- Demonstrate how Codex helped convert the project vision into a working solution.

## Track

Voice & Accountability: tools that strengthen civic participation, hold power to account, and bridge the gap between citizens and the institutions that serve them.
